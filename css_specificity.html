book CSS Specificity 

            CSS Specificity เป็นเหมือนค่าที่ทำให้หน้าเว็บเราเลือกว่าจะใช้ Style อันไหน
            ซึ่งมีหลักการคือถ้าค่าไหนมากกว่าก็นำค่านั้นมาแสดงผล และวิธีการคำนวณค่าตรงนี้
            คือ เราจะสมมุติค่าใน Style ขึ้นมาดังนี้
            - Inline Style มีค่า 1000
            - ID selector ทีค่า 100
            - Class selector มีค่า 10
            - All tag html มีค่า 1
            ทีนี้ถ้ามีค่าพวกนี้ซ้อนกันหลายชั้น เราจำนำค่าพวกนี้มาบวกกัน เช่น
             มีค่า = 1(body)+1(h1) = 2
             มีค่า = 100(#red)+10(.box) = 110
             มีค่า = 1(div)+10(.box) = 11

            เพื่อพิสูจน์ เราได้ยกตัวอย่าง code ง่ายๆ 
            ว่าสิ่งที่เขียนไปข้างบนมีการเลือกค่าที่มากกว่ามาแสดงผลจริงๆ คือ

            ตัวอย่างที่ 1
            
            ผลลัพธ์ที่ได้
            
            จะเห็นได้ว่า กล่องที่ออกมาเขียวสีเขียว ถึงจะเขียนทับไปแล้วว่าให้เป็นสีแดง
            เพราะว่า ส่วนที่เป็น #green มีค่าถึง 100 แต่ .red มีค่าแค่ 10 เท่านั้นเอง
            จึงไม่สามารถเอาชนะค่า #green ได้ จึงทำให้กล่องไม่ออกมาเป็นสีแดงถึงแม้จะถูกทับแล้วก็ตาม

            ตัวอย่างที่ 2
            
            ผลลัพธ์ที่ได้
            
            จะเห็นได้ว่า กล่องเป็นสีแดง เพราะ div.box มีค่าเป็น 11(10+1)
            .box มีค่า 10 น้อยกว่า div.box จึงทำให้กล่องเป็นสีแดง
            
            ถ้าจะให้สรุปหลักการของ CSS Specificity ง่ายๆด้วยเหตุและผลก็คือ ยิ่งเขียน Style
            ให้เจาะจงมากก็ยิ่งทำให้มีโอกาสได้แสดงมาก
